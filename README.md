# Recon Automation Tool

## Overview
This project demonstrates a basic reconnaissance automation script used in cybersecurity assessments and SOC investigations. The tool automates common recon tasks and generates a structured report for analyst review.

## Features
- DNS to IP resolution
- Common port scanning
- WHOIS information gathering
- Automated report generation

## Tools & Technologies
- Python
- Socket Programming
- Command-line execution

## How It Works
The user provides a target domain. The script performs reconnaissance and saves the results in a report file for analysis.

## Output
- recon_report.txt (structured reconnaissance findings)

This project is strictly for educational and ethical learning purposes.
